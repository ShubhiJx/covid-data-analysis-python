# COVID-19 Data Analysis using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on COVID-19 data using Python.  
The aim is to understand trends in cases and deaths and compare data between countries using visualizations.

## Dataset
The dataset contains COVID-19 statistics such as:
- total cases
- new cases
- total deaths
- date
- country name

File used:
covid_small.csv

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

1. Country Comparison
Compared COVID-19 total cases between India and USA.

2. Moving Average
Calculated 7-day moving average to understand trend clearly.

3. Death Trend Analysis
Visualized increase in deaths over time.

4. Bar Chart Visualization
Compared total cases across multiple countries.

5. Heatmap Visualization
Displayed intensity of cases using color scale.

6. Summary Statistics
Used groupby to find maximum cases and deaths country-wise.

## Key Insights
- COVID cases increased rapidly during mid period.
- Moving average helps remove daily fluctuations.
- Some countries had significantly higher case counts.
- Death trends follow similar pattern as total cases.
