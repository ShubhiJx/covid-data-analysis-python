# COVID-19 Data Analysis using Python

## Overview
This project performs Exploratory Data Analysis (EDA) on COVID-19 dataset using Python.  
The goal is to understand trends in cases, deaths and compare countries using data visualization.

## Dataset
The dataset contains COVID-19 information such as:
- total cases
- new cases
- total deaths
- date
- country name

File used:
covid_small.csv

## Technologies Used
- Python
- Pandas
- Matplotlib
- NumPy
- Jupyter Notebook

## Visualizations Included

1. India vs USA Comparison Graph
Comparison of total COVID cases between India and United States.

2. Moving Average Trend
3-day moving average used to smooth daily case fluctuations.

3. Death Trend Graph
Shows increase in total deaths over time.

4. Bar Chart of Total Cases
Comparison of total cases across countries.

5. Heatmap Visualization
Color-based visualization showing intensity of cases.

6. Correlation Heatmap
Shows relationship between cases and deaths.

1. Download repository
2. Open notebook in Jupyter Notebook or VS Code
3. Run all cells
4. Graphs will be generated

## Key Insights
- COVID cases show increasing trend over time
- Moving average helps understand pattern clearly
- Some countries have significantly higher cases
- Deaths trend is correlated with total cases
